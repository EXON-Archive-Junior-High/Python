# 연산
파이썬에서 더하기, 빼기, 나누기, 곱하기, 나머지 연산을 해봅시다! <br>
+는 더하기, -는 빼기, *는 곱하기, /는 나누기입니다. <br>


## 코드
```python
print(1 + 2) #더하기 이므로 3이 출력
print(5 - 3) #빼기 이므로 2가 출력
print(3 * 2) #곱하기 이므로 6이 출력
print(6 / 3) #나누기 이므로 2가 출력
```

## 예제
사용자가 입력한 수를 더해주는 프로그램
```python
a = (int)input()
b = (int)input()
print(a + b)
```
int()는 숫자형태를 받아온다는 뜻입니다.
